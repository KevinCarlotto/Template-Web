Template Web - Base Web

A simple web template designed as a starting point for developing websites.

---

## Overview

This project provides a basic web structure to help guide the development of websites.

It can be used as:

- A learning base for beginners
- A starting template for web projects
- A structure for local development and testing

⚠️ This is not a production-ready system, but a foundation for development and experimentation.

---

## Usage

This template can be used as a base for building web pages.

To use it:

1. Download or clone the repository
2. Place it in your local server (XAMPP, Laragon, or similar)
3. Open it in your browser
4. Modify the structure according to your needs

You can use it for:
- Learning web development
- Creating custom websites
- Testing layouts and ideas
